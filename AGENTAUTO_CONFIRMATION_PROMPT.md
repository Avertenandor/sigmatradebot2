# ПРЕДВАРИТЕЛЬНОЕ ПОДТВЕРЖДЕНИЕ ПОНИМАНИЯ ЗАДАЧИ

## Инструкция для AgentAuto

Перед началом выполнения основного техзадания из файла `AGENTAUTO_DEPLOY_PROMPT.md`, выполни следующее:

### ШАГ 1: ПРОЧИТАЙ И ПЕРЕСКАЖИ

1. Прочитай полностью файл `AGENTAUTO_DEPLOY_PROMPT.md`.

2. В своём ответе кратко перескажи своими словами:
   - Какие основные проблемы нужно решить?
   - Какие этапы работы предстоит выполнить?
   - Какие файлы будут изменяться?
   - Какие проверки нужно будет провести?

3. Укажи, есть ли что-то неясное в инструкциях, или ты готов приступить к выполнению.

### ШАГ 2: ПОДТВЕРЖДЕНИЕ ГОТОВНОСТИ

После пересказа напиши:

```
✅ Я понял задачу и готов приступить к выполнению этапов 0-6.
```

Или, если есть вопросы:

```
❓ У меня есть вопросы по [указать что именно].
```

### ШАГ 3: НАЧАЛО РАБОТЫ

Только после подтверждения понимания начинай выполнение этапов из `AGENTAUTO_DEPLOY_PROMPT.md`.

---

**ВАЖНО:** Не начинай выполнение команд до тех пор, пока не получишь подтверждение от пользователя, что твой пересказ корректен.
